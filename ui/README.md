- `IndicatorPanel.qml` — НОВЫЙ компонент (заменяет WarningStrip в центре)
- `Dashboard.qml` — обновлён (IndicatorPanel + Display вместо WarningStrip + Display)

- `main.qml` — без изменений
- `qmldir` — без изменений

## Как интегрировать

1. Скопируй `Display.qml`, `IndicatorPanel.qml`, `Dashboard.qml`
   в папку `ui/` проекта (заменив существующие файлы)

2. `IndicatorPanel.qml` — новый файл, добавь его в `resources.qrc`:
   ```xml
   <file>ui/IndicatorPanel.qml</file>
   ```

3. `WarningStrip` больше не вызывается из Dashboard — можно оставить файл,
   он просто не будет использоваться.

## Структура центральной зоны (v3)

```
centerZone (anchors: между спидометром и тахометром)
├── IndicatorPanel (top, height: 40%)
│   ├── criticalZone — МАСЛО / ПЕРЕГРЕВ / ТОРМОЗ / АКБ / ПОДУШКИ / РЕМЕНЬ
│   ├── warningZone  — CHECK / ABS / ESP / TPMS / ТОПЛИВО / КОЛОДКИ / РУЛЕВОЙ
│   └── infoZone     — БЛИЖНИЙ / ДАЛЬНИЙ / ТУМАН / КРУИЗ
└── Display (bottom, height: 60%)
    ├── headerZone — ВРЕМЯ (слева) / ENGINE STATUS (центр) / ДАТА (справа)
    ├── centerZone — СКОРОСТЬ (большая) / ПЕРЕДАЧА / КРУИЗ / ПОВОРОТНИКИ
    └── footerZone — ПРОБЕГ (слева) / ТЕМПЕРАТУРА (центр) / РАСХОД (справа)
```

## Ключевые улучшения

### Индикаторы (IndicatorPanel):
- Три чётко разделённые зоны: критические / предупреждения / информационные
- Каждый индикатор — «плитка» с иконкой + подписью (не просто иконка)
- Критические: крупнее (+20%), с пульсирующим свечением
- Неактивные: полупрозрачные 7.5% — занимают место, не «прыгают»
- Self-test при старте: все горят 3 сек, затем гаснут кроме активных

### Display:
- **Header**: время hh:mm слева крупно, день недели под ним; дата dd.MM.yyyy справа
- **Center**: большая скорость центрирована; передача и CC в строке над
- **Footer**: пробег слева | температура с баром по центру | расход справа
- Температура цифровая (°C) с цветовой индикацией: синий → зелёный → жёлтый → красный
- Расход меняет цвет: норма → предупреждение → критично

### Анимации:
- ColorAnimation для всех цветовых переходов
- NumberAnimation для температурного бара (600мс)
- Пульсация критических индикаторов (500мс cycle)
- Fade-in при старте через opacity (сохранено из v2)
